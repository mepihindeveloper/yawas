# /runtime - переменные/изменяемые файлы

Этот каталог содержит часто изменяющиеся файлы с нефиксированным размером. Примерами таким файлов могут служить логи, 
кеши, базы данных и прочие.

## Файловая структура

```
/runtime: переменные/изменяемые файлы
    - /log: файлы логов
    - /cache: файлы кеша
```
